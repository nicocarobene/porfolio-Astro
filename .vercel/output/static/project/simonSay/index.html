Quick Selector

|

Quick Selector is a Visual Studio Code extension that provides tools to ease your programming workflow. Simply select the code for which you want to generate CSS selectors in your HTML file. Then, access the class creator using Ctrl + Shift + P, type 'Generate CSS', and voila! You'll have all your CSS selectors in the clipboard ready to paste into your .css file
